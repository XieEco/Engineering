This code establishes a simulation framework for the electric vehicle—power grid—coal power plant, capturing the electricity supply chain and associated carbon and air pollutant emission transfer, using vehicle-level and plant-level data.  

After extracting "Data.zip" to the same folder as "ENG0818.m", run it using MATLAB.  
The recommended environment is MATLAB R2016b.
